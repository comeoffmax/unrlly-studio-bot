# unrlly Studio Bot — внутренний инструмент

Приватный бот только для команды (белый список по Telegram ID).

---

## Как работает

### Режим /brief — сбор брифа
1. Пишешь `/brief`
2. Вставляешь первое сообщение от клиента (дословно или пересказом)
3. Бот анализирует и даёт **один уточняющий вопрос** — берёшь, задаёшь клиенту
4. Вставляешь ответ клиента → бот задаёт следующий вопрос
5. Когда собрано всё нужное — бот выдаёт структурированный бриф
6. Кнопка **"Генерировать КП"** прямо из брифа

### Режим /kp — генерация КП
- Если бриф уже собран → сразу генерирует
- Если нет → просит вставить данные вручную
- В КП место для цены помечено `[ЦЕНА]` — заменяешь вручную перед отправкой клиенту

### Свободный режим
Без команды — просто пишешь вопрос, бот отвечает с учётом контекста текущего проекта.

## Деплой на Railway

### 1. Создай бота
[@BotFather](https://t.me/BotFather) → `/newbot` → получи токен

### 2. Узнай свой Telegram ID
Напиши [@userinfobot](https://t.me/userinfobot) — он ответит твоим ID.
Для Кати — она тоже пишет @userinfobot.

### 3. Задеплой

```bash
npm install -g @railway/cli
railway login

cd studio_bot
railway init   # Empty project
railway up
```

### 4. Переменные окружения в Railway Dashboard → Variables

| Переменная | Значение |
|---|---|
| `TELEGRAM_TOKEN` | токен от BotFather |
| `ANTHROPIC_API_KEY` | ключ с console.anthropic.com |
| `ALLOWED_USER_IDS` | `123456789,987654321` (ID Макса и Кати через запятую) |

## Типичный сценарий работы

```
Ты: /brief

Бот: Вставляй сообщения от клиента...

Ты: [вставляешь] "Нам нужен сайт для нашего ивент-агентства, 
     хотим современный дизайн, есть референсы"

Бот: ❓ Уточняющий вопрос для клиента:
     "Какой главный результат от сайта — заявки, имидж, 
     что-то другое?"

Ты: [спрашиваешь клиента, вставляешь ответ]
    "В первую очередь заявки на организацию мероприятий"

Бот: ❓ Сколько экранов примерно? 
     Лендинг (5–7) или многостраничный сайт?

... (ещё 2–3 вопроса)

Бот: ## 📋 Бриф — ивент-агентство
     Тип: Site Plus
     Задача: ...
     [полный бриф]
     
     ✅ Бриф готов!
     [кнопка: Генерировать КП]

Ты: [нажимаешь кнопку]

Бот: ## Коммерческое предложение...
     Стоимость: [ЦЕНА]  ← заменяешь своей цифрой
     
     ⚠️ Замени [ЦЕНА] перед отправкой клиенту
```
